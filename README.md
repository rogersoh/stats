It display the current ip address, the CPU temperature

Raspberry Pi
OLED 128x32
INA21
Breadboard

Connection
Raspberry Pi

Ground   OLED, INA21 Ground
GPIO 14  OLED RST
SCL      OLED, INA21 SCL
SDA      OLED, INA21 SDA
